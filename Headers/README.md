This folder contains the mapping between the attributed internal codes and the original sequence headers.

Headers retrieved from different databases are organized in the dataframes provided here. The format of each header and its corresponding database of origin are listed below.

fig|XXX.XXXX.peg.XXXX — BV-BRC database  
Example: fig|487.3834.peg.1801  

tr/sp|XXXX|XXXX — UniProt database  
Example: tr|A0A023CJ62|A0A023CJ62_9BACI  

XXXXXXXX.X — RefSeq database  
Example: EEY13283.1  

abc:XXXXXX — KEGG database  
Example: ecm:EcSMS35_3228  

gene.aXX — dbCAN-PUL database  
Example: neuS.p01  

MetaCyc and CAZy databases report UniProt and RefSeq headers.

For some internal codes, more than one header may be associated.
